## Про курс

Цей курс допоможе тобі опанувати TypeScript — типізовану надбудову над JavaScript, яка робить код надійнішим та зрозумілішим.

### Що ти вивчиш:

- Базові та складні типи даних
- Інтерфейси та типи
- Класи та ООП в TypeScript
- Generics для гнучкого коду
- Модульну систему
- Налаштування проєктів

### Для кого цей курс:

- Розробники, які знають JavaScript і хочуть вивчити TypeScript
- Початківці, які хочуть писати надійніший код
- Всі, хто хоче покращити свої навички
